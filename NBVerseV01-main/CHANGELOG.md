# Changelog

## v0.2.1 (2024-12-21)

### Added
- 컴팩트 저장소 기능 (`NBverseCompactStorage`)
  - 단일 JSON 파일에 최대 25개 데이터 유지
  - FIFO 방식: 새 데이터 추가 시 가장 오래된 데이터 자동 제거
  - 타임스탬프 자동 생성
  - 히스토리 자동 기록 (최대 100개)
  - 편의 함수: `add_text_compact()`
- 파일 크기 최적화: 약 10-15 KB (25개 기준)

### Changed
- `input_view.py`: 컴팩트 저장소 통합
- 저장 시 자동으로 타임스탬프 생성
- 히스토리 자동 기록

## v0.2.0 (2024-12-21)

### Added
- 유사도 검색 기능
  - N/B 값 기반 유사도 계산
  - 텍스트 기반 유사도 계산 (Jaccard, Levenshtein)
  - 하이브리드 유사도 (N/B + 텍스트 가중 평균)
- 조회 히스토리 및 타임라인
  - 조회 기록 자동 저장
  - 타임라인 조회 기능
  - 조회 통계 제공
- 입력/조회 도구 개선
  - 정확 일치 → 유사도 검색 → 저장 순서
  - 유사도 결과 표시
  - 최근 조회 타임라인 표시

### Changed
- `input_view.py`: 유사도 검색 및 히스토리 저장 기능 추가
- `storage.py`: 범위 검색 기능 추가 (`find_similar_by_nb_range`)

## v0.1.0 (2024-12-21)

### Added
- N/B 값 계산 기능 (bitCalculation.v.0.2.js 기반)
- 텍스트를 N/B 값으로 변환
- max/min 폴더 구조로 데이터 저장
- CLI 도구 및 입력/조회 기능
- 설정 관리 (소수점 자리수 등)
- 배치 파일로 실행 가능
